# Tomb Raider Shortcut Maker

WinUI 3 utility to create Windows shortcuts for classic Tomb Raider PC games.

## Features
- Create shortcuts for TR CD Editions
- Optional launch arguments (-gold, -setup)
- Windows 11 native UI (Mica)
- Simple and fast

## Build
Requires:
- Visual Studio 2022
- .NET 8
- Windows App SDK
